This platform is a full-featured blog application built with Django, designed to provide a seamless experience for users to share their thoughts, ideas, and stories.

Key Features

{% endcomment %}

Technical Implementation

  • Built with Django framework for robust back-end functionality
  • Uses PostgreSQL from Supabase as backend database service
  • AWS S3 integration for scalable file and image storage
  • Implemented content moderation using Hugging Face AI models to detect profanity in posts and images
  • Class-based views for efficient code organization
  • Django's built-in authentication system with customizations
  • Created user profile pages with credential management and profile picture customization
  • Responsive front-end design with modern CSS

User Management

  • Secure user registration and authentication
  • Personalized user profiles with customizable images
  • Profile management and updates

Content Creation & Management

  • Create, edit, and delete blog posts
  • Rich text formatting for blog content
  • Content moderation with profanity filtering
  • Image upload with content safety verification

User Experience

  • Mobile-responsive design
  • Pagination for easy navigation through content
  • User-specific content views to browse posts by author
  • Intuitive interface for both readers and content creators
